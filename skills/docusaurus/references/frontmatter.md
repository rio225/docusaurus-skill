## Doc Frontmatter

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `title` | string | Filename | Page title (h1 and browser tab) |
| `description` | string | First paragraph | Meta description for SEO |
| `sidebar_label` | string | `title` | Shorter label for sidebar display |
| `sidebar_position` | number | Alpha order | Position in autogenerated sidebars |
| `sidebar_class_name` | string | - | Custom CSS class for sidebar item |
| `slug` | string | File path | Custom URL path (e.g., `/custom-url`) |
| `tags` | array | `[]` | Tags for categorization |
| `keywords` | array | `[]` | SEO keywords for meta tag |
| `image` | string | - | OG image for social sharing |
| `hide_title` | boolean | `false` | Hide the h1 heading |
| `hide_table_of_contents` | boolean | `false` | Hide right-side TOC |
| `toc_min_heading_level` | number | `2` | Min heading level in TOC |
| `toc_max_heading_level` | number | `3` | Max heading level in TOC |
| `pagination_next` | string\|null | Auto | Override next doc link |
| `pagination_prev` | string\|null | Auto | Override previous doc link |
| `pagination_label` | string | `title` | Label shown in pagination |
| `draft` | boolean | `false` | Exclude from production build |
| `unlisted` | boolean | `false` | Hide from sidebar, accessible via URL |
| `last_update` | object | Git data | `{date: 'YYYY-MM-DD', author: 'Name'}` |
| `custom_edit_url` | string\|null | Auto | Custom "Edit this page" link |
| `parse_number_prefixes` | boolean | `true` | Parse `01-intro.md` as position 1 |

### Complete Doc Example

```yaml
---
title: "Authentication Guide"
description: "Learn how to implement secure user authentication with JWT tokens and OAuth2 integration."
sidebar_label: "Auth Guide"
sidebar_position: 3
slug: /guides/authentication
tags:
  - security
  - authentication
  - tutorial
keywords:
  - JWT
  - OAuth2
  - login
  - security
image: /img/guides/auth-og.png
hide_table_of_contents: false
toc_min_heading_level: 2
toc_max_heading_level: 4
pagination_next: guides/authorization
pagination_prev: guides/getting-started
draft: false
last_update:
  date: 2024-01-15
  author: Vipul Gupta
---
```

## Blog Frontmatter

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `title` | string | Required | Post title |
| `description` | string | Truncated content | Meta description |
| `slug` | string | Filename date | Custom URL slug |
| `date` | string | Filename/git | Post date (ISO format) |
| `authors` | array\|object | - | Author(s) - keys or inline |
| `tags` | array | `[]` | Blog tags for filtering |
| `image` | string | - | OG image for sharing |
| `hide_table_of_contents` | boolean | `false` | Hide TOC |
| `draft` | boolean | `false` | Exclude from build |
| `unlisted` | boolean | `false` | Hide from list, keep URL |
| `truncate_marker` | string | `<!--truncate-->` | Custom truncate marker |

### Authors Configuration

**Method 1: Global authors.yml**

Create `blog/authors.yml`:
```yaml
vipul:
  name: Vipul Gupta
  title: Developer & Community Builder
  url: https://github.com/vipulgupta2048
  image_url: https://github.com/vipulgupta2048.png
  email: vipulgupta2048@gmail.com
  socials:
    twitter: vipulgupta2048
    github: vipulgupta2048

alice:
  name: Alice Smith
  title: Technical Writer
  url: https://example.com/alice
  image_url: /img/authors/alice.png
```

Reference in frontmatter:
```yaml
authors: [vipul]
# Multiple authors:
authors: [vipul, alice]
```

**Method 2: Inline authors**
```yaml
authors:
  - name: Guest Author
    title: External Contributor
    url: https://example.com
    image_url: https://example.com/avatar.png
```

**Method 3: Mixed**
```yaml
authors:
  - vipul  # From authors.yml
  - name: Guest Author
    title: One-time contributor
```

## Page Frontmatter

For pages in `src/pages/*.md` or `src/pages/*.mdx`:

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `title` | string | - | Page title |
| `description` | string | - | Meta description |
| `wrapperClassName` | string | - | Wrapper div class |
| `hide_table_of_contents` | boolean | `true` | Show/hide TOC |

### Page Example

```yaml
---
title: About Us
description: Learn about our team and mission
wrapperClassName: about-page
---

# About Us

Custom page content...
```

## Special Values

### Disabling Pagination
```yaml
pagination_next: null
pagination_prev: null
```

### Custom Edit URL
```yaml
# Disable edit link
custom_edit_url: null

# Custom URL
custom_edit_url: https://github.com/org/repo/edit/main/custom/path.md
```

### Relative Image Paths
```yaml
# From static folder
image: /img/og-image.png

# External URL
image: https://example.com/image.png
```
